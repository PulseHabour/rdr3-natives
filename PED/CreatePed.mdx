## CREATE_PED

```c
// 0xD49F9B0955C367DE
Ped CREATE_PED(Hash modelHash, Vector3 pos, float heading, BOOL isNetwork, BOOL bScriptHostPed, BOOL p7, BOOL p8);
```


## Parameters
* **modelHash**: The model to load, you must <Native name="REQUEST_MODEL" /> before calling this or it will fail.
* **pos**: The position to spawn the ped at
* **heading**: The heading at which to spawn the ped
* **isNetwork**: `true` if you want the ped to be networked, `false` if you want it to be a local ped
* **bScriptHostPed**: If we should bind the ped to a script.
* **p7**:
* **p8**:

## Return value
Returns the ped handle for the ped, or `0` if the ped failed to spawn
