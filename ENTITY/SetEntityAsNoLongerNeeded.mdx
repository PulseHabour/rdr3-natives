## SET_ENTITY_AS_NO_LONGER_NEEDED

```c
// 0x4971D2F8162B9674
void SET_ENTITY_AS_NO_LONGER_NEEDED(Entity* entity);
```

Marks the specified entity (ped, vehicle or object) as no longer needed, allowing the game engine to delete it whenever it sees fit.

## Parameters
* **entity**: The entity to be marked as no longer needed
